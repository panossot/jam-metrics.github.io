JBoss-Automated-Metrics Repo

JBoss-Automated-Metrics library is nowadays hosted on github repo https://github.com/panossot/jboss-automated-metrics. The repo contains various subfolders. Each of them adds to JBoss-Automated-Metrics some functionality :

  • AutomatedMetricsStandalone : Contains the code used for Wildfly Standalone version (plus test cases).
  • AutomatedMetricsSubSystemModules : Contains the code used for Wildfly subsystem version (plus test cases).
  • JavaSeAutomatedMetrics : Contains the code used for Java SE version (plus test cases).
  • StandardMetricsReadyForUsage : Contains basic examples to demonstrate how JBoss-Automated-Metrics could be used to monitor JVM Memory, Disk Space, etc.
  • MetricsRhqPlugin : Contains the plugin to be modified and uploaded on Rhq Server in order to monitor the selected metrics.
  • WildflyAddExtensionSubsystem/AutomatedMetricsSubsystemExtension : Contains the plugin to be executed in order to load JBoss-Automated-Metrics Subsystem on Wildfly.
