JAM-Metrics Projects

JAM-Metrics are divided in three projects :

  1. A Standalone Library used with any deployment on Wildfly Server.
  2. An optional Subsystem of Wildfly.
  3. A Library used with any Java SE application.

 

Each project uses its own artifacts :

Standalone Library used with any deployment on Wildfly Server
  • org.jam.metrics.ApplicationMetrics
  • org.jam.metrics.ApplicationMetricsApi
  • org.jam.metrics.ApplicationMetricsProperties
Optional Subsystem of Wildfly
  • org.jam.metrics.ApplicationMetricsApi
  • org.jam.metrics.ApplicationMetricsProperties
Library used with any Java SE application
  • org.jam.metrics.ApplicationMetricsJavaSeApi
  • org.jam.metrics.ApplicationMetricsProperties
